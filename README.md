


## AWS datacenter

Amazon AWS datacenter that runs EC2 instances with Linux 2 os. Every instance runs a Docker service:
Nginx server, Sinatra server, Redis database, Jenkins pipeline.


## Delete the datacenter

```
cd aws-datacenter
amazon/run/delete.sh

```
